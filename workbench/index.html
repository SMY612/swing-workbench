วิธีอ่านจอนี้ (สำหรับมือใหม่ — แตะเพื่อย่อ/ขยาย)
  1. ไฟกลมบนหัวจอ = สภาพตลาด: เขียว = เปิดรับ เข้าไม้ได้ · แดง = ห้ามเปิดไม้ใหม่
  2. แถบเตือนใต้หัวจอ: แดง = ห้าม/อันตราย · เหลือง = ระวัง · เขียว = โอกาส · ฟ้า = ข้อมูลทั่วไป
  3. ชิปตัวกรอง (ทั้งหมด / เข้าเกณฑ์ / ใกล้ / ติดงบ): แตะเพื่อกรองรายชื่อหุ้น
  4. แตะหุ้น 1 ตัว → กราฟ + การ์ดระดับราคา + เช็คลิสต์ 7 ข้อของตัวนั้นจะแสดง (มือถือ: อยู่ใต้รายการ)
  5. เช็คลิสต์ 7 ข้อ: ระบบเช็คให้อัตโนมัติตามกติกา v2 — ผ่านครบ = เข้าเกณฑ์ breakout ไม่ต้องคำนวณเอง

ทุกตัวเลขในจอนี้มาจากราคาปิดของเมื่อคืน — ตัดสินใจวันละครั้ง ไม่ไล่ราคาระหว่างวัน
